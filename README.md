## Build Setup

``` bash
# install dependencies
npm(yarn) install

# start the api (&production test) server
npm run start

# serve with hot reload at localhost:8080 (Some APIs won't work)
npm run dev

# build for production with minification
npm run prod

# run unit tests
npm run unit

# run all tests
npm test
```
